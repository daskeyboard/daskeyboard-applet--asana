# Q Applet: Asana

Displays Asana notifications from your prenium account on a Das Keyboard Q Series.

Example:

![Asana on a Das Keyboard Q](assets/image.png "Q Asana")

## Changelog

[CHANGELOG.MD](CHANGELOG.md)

## Installation

Requires a Das Keyboard Q Series: www.daskeyboard.com and a Asana account.

Installation, configuration and uninstallation of applets is done within
the Q Desktop application (https://www.daskeyboard.com/q)

## Running tests

- `yarn test`
